На каком уровне детализации строим Gantt?

Три варианта. Нажмите на вкладку, чтобы посмотреть mockup в полном размере. Когда определитесь — нажмите «Выбрать этот вариант».

A — Портфельный Gantt
Одна строка = одна инициатива. Быстрый обзор портфеля за квартал. Ресурсная нагрузка — агрегировано по ролям понедельно. Подходит для планёрок с руководством и защиты квартального плана.

B — Двухуровневый Gantt
Инициативы можно раскрыть и увидеть эпики внутри. Стрелки зависимостей между эпиками. Баланс между обзором и детализацией — подходит для еженедельных статусных встреч команды.

C — Полный task-level Gantt
Полная 4-уровневая иерархия: Инициатива → Эпик → Story → Подзадача. Критический путь выделен красным. Полезен для точного планирования спринтов, но требует постоянной актуализации.
